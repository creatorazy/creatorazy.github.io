Java如何复制一个数组

AZy

2019-07-11

 

Java

本次要跟大家讨论一下在Java中如何复制数组，以及常见的错误

新手常见的错误

1.直接把一个数组赋值给另一个数组

给一个数组赋值（错误示例）

这是很多新手犯得错误，这样的写法其实是把arr数组的地址赋值给arr1,相当于arr，arr1指向的是同一个地址，虽然这样会让他们的元素值相等，但是arr元素修改时，arr1对应的元素也会修改，因为他们是指向同一个地址。

数组的正确赋值方式

java 提供了一个Arrays类，这个类包含了两个方法

1.Arrays.copyOf(arr,newLength)

arr参数代表要复制的数组
newLength参数代表要复制的长度
例

这样就相当于把arr数组所有元素都复制给arr1数组了

注意:
如果newLength参数比被复制数组长度长的话，则会在后面补0
例

则 arr1 的 元素分别 1， 2 ，3 ， 0

2.Arrays.copyOfRange(arr,fromIndex,toIndex)

arr参数代表被复制数组
fromIndex参数代表复制起始下标
toIndex参数代表结束下标(不复制结束下标)
例

这样就相当于把arr数组里面下标从1开始到下标3(因为不复制结束下标4)的元素复制给arr1数组了
所以 arr1 的 元素 分别是 2, 3, 4
注意:
toIndex参数的值 和 fromIndex 参数值相同不会报错，但是不会复制任何元素(无意义操作)
如果toIndex - 1 超过了 arr 数组长度，则用0代表被复制数
例

则arr1 数组元素分别是1,2,3,4,0
